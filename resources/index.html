To install Platypuslol, just do following:
  • If you are on Firefox browser, click here to add it. I recommend to add it as your default search engine, so that you have search suggestions. Otherwise, go to settings for search engines, and assign keyword to it (for example "p") and that as prefix for your searches.
  • If you are using Google Chrome, do following.
    1. Make sure you navigated to the domain url {server} (not "{server}/install").
    3. Platypus search engine should be added to your list of search engines. Go to settings and set it as default (or set proper keyword, like "p").
  • If you are on other browser, and none of the above works, try to install following opensearch plugin manually in your browser, or create search plugin with following search url "{server}/redirect?q={searchTerms}" and following suggestion url "{server}/suggest?q={searchTerms}".
